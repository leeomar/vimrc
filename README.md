yuxio's vimrc
=============

Author: Tzeng Yuxio <tzengyuxio@gmail.com>

Web Site: https://github.com/tzengyuxio/vimrc


Install
-------

* Mac OS X

```sh
git clone https://github.com/tzengyuxio/vimrc ~/.vim
ln -s ~/.vimrc ~/.vim/vimrc
cd ~/.vim
git submodule update --init
```


Key Mappings
------------

* F6: switch search hilight
